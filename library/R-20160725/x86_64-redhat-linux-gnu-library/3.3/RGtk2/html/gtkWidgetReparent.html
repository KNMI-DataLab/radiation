gtkWidgetReparent

Description

Moves a widget from one GtkContainer to another, handling reference count issues to avoid destroying the widget.

Usage

gtkWidgetReparent(object, new.parent)

Arguments

object

a GtkWidget

new.parent

a GtkContainer to move the widget into
