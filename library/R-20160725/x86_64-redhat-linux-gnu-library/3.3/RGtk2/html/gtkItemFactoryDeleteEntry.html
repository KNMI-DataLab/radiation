gtkItemFactoryDeleteEntry

Description

Deletes the menu item which was created from entry by the given item factory. WARNING: gtk_item_factory_delete_entry has been deprecated since version 2.4 and should not be used in newly-written code. Use GtkUIManager instead.

Usage

gtkItemFactoryDeleteEntry(object, entry)

Arguments

object

a GtkItemFactory

entry

a GtkItemFactoryEntry
