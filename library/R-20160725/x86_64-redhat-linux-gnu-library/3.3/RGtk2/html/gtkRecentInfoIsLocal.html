gtkRecentInfoIsLocal

Description

Checks whether the resource is local or not by looking at the scheme of its URI.

Usage

gtkRecentInfoIsLocal(object)

Arguments

object

a GtkRecentInfo

Details

Since 2.10

Value

[logical] TRUE if the resource is local.
