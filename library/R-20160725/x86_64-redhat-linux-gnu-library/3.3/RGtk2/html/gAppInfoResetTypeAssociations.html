gAppInfoResetTypeAssociations

Description

Removes all changes to the type associations done by gAppInfoSetAsDefaultForType, gAppInfoSetAsDefaultForExtension, gAppInfoAddSupportsType or gAppInfoRemoveSupportsType.

Usage

gAppInfoResetTypeAssociations(content.type)

Arguments

content.type

a content type
