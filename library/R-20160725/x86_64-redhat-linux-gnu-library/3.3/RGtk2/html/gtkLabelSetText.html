gtkLabelSetText

Description

Sets the text within the GtkLabel widget. It overwrites any text that was there before.

Usage

gtkLabelSetText(object, str)

Arguments

object

a GtkLabel

str

The text you want to set

Details

This will also clear any previously set mnemonic accelerators.
