gdkRegionUnion

Description

Sets the area of source1 to the union of the areas of source1 and source2. The resulting area is the set of pixels contained in either source1 or source2.

Usage

gdkRegionUnion(object, source2)

Arguments

object

a GdkRegion

source2

a GdkRegion
