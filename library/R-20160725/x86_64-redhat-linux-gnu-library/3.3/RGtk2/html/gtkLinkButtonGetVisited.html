gtkLinkButtonGetVisited

Description

Retrieves the 'visited' state of the URI where the GtkLinkButton points. The button becomes visited when it is clicked. If the URI is changed on the button, the 'visited' state is unset again.

Usage

gtkLinkButtonGetVisited(object)

Arguments

object

a GtkLinkButton

Details

The state may also be changed using gtkLinkButtonSetVisited. Since 2.14

Value

[logical] TRUE if the link has been visited, FALSE otherwise
