gtkLabelGetLineWrapMode

Description

Returns line wrap mode used by the label. See gtkLabelSetLineWrapMode.

Usage

gtkLabelGetLineWrapMode(object)

Arguments

object

a GtkLabel

Details

Since 2.10

Value

[PangoWrapMode] TRUE if the lines of the label are automatically wrapped.
