gtkRcGetImModuleFile

Description

Obtains the path to the IM modules file. See the documentation of the GTK_IM_MODULE_FILE environment variable for more details.

Usage

gtkRcGetImModuleFile()

Value

[character] a newly-allocated string containing the name of the file listing the IM modules available for loading
