gtkBuildableAddChild

Description

Adds a child to buildable. type is an optional string describing how the child should be added.

Usage

gtkBuildableAddChild(object, builder, child, type)

Arguments

object

a GtkBuildable

builder

a GtkBuilder

child

child to add

type

kind of child or NULL. [ allow-none ]
