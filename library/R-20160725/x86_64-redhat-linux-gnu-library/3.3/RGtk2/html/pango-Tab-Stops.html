Detailed Description

Functions in this section are used to deal with PangoTabArray objects that can be used to set tab stop positions in a PangoLayout.

Structures

PangoTabArray

A PangoTabArray struct contains a list of tab stops. Each tab stop has an alignment and a position.

Enums and Flags

PangoTabAlign

A PangoTabAlign specifies where a tab stop appears relative to the text.

left

the tab stop appears to the left of the text.
