gdkDrawableGetData

Description

Equivalent to gObjectGetData; the GObject variant should be used instead. WARNING: gdk_drawable_get_data is deprecated and should not be used in newly-written code.

Usage

gdkDrawableGetData(object, key)

Arguments

object

a GdkDrawable

key

name the data was stored under

Value

[R object] the data stored at key
