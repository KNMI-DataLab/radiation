cairoUserFontFaceSetTextToGlyphsFunc

Description

Sets th text-to-glyphs conversion function of a user-font. See cairo_user_scaled_font_text_to_glyphs_func_t for details of how the callback works.

Usage

cairoUserFontFaceSetTextToGlyphsFunc(font.face, text.to.glyphs.func)

Arguments

font.face

[CairoFontFace] A user font face

text.to.glyphs.func

[cairo_user_scaled_font_text_to_glyphs_func_t] The text_to_glyphs callback, or NULL

Details

The font-face should not be immutable or a CAIRO_STATUS_USER_FONT_IMMUTABLE error will occur. A user font-face is immutable as soon as a scaled-font is created from it. Since 1.8
