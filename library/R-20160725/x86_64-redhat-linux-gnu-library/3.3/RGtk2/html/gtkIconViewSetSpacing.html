gtkIconViewSetSpacing

Description

Sets the ::spacing property which specifies the space which is inserted between the cells (i.e. the icon and the text) of an item.

Usage

gtkIconViewSetSpacing(object, spacing)

Arguments

object

a GtkIconView

spacing

the spacing
