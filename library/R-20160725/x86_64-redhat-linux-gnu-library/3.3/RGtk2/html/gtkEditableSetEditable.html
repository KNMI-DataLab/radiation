gtkEditableSetEditable

Description

Determines if the user can edit the text in the editable widget or not.

Usage

gtkEditableSetEditable(object, is.editable)

Arguments

object

a GtkEditable

is.editable

TRUE if the user is allowed to edit the text in the widget
