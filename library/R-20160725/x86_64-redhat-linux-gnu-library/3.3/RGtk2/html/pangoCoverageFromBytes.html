pangoCoverageFromBytes

Description

Convert data generated from pangoConverageToBytes() back to a PangoCoverage

Usage

pangoCoverageFromBytes(bytes)

Arguments

bytes

[raw] binary data representing a PangoCoverage

Value

[PangoCoverage] a newly allocated PangoCoverage, or NULL if the data was invalid.
