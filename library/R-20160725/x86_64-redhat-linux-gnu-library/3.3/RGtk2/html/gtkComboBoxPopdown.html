gtkComboBoxPopdown

Description

Hides the menu or dropdown list of combo.box.

Usage

gtkComboBoxPopdown(object)

Arguments

object

a GtkComboBox

Details

This function is mostly intended for use by accessibility technologies; applications should have little use for it. Since 2.4
