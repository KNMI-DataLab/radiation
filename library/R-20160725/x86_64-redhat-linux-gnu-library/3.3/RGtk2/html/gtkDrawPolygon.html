gtkDrawPolygon

Description

Draws a polygon on window with the given parameters. WARNING: gtk_draw_polygon has been deprecated since version 2.0 and should not be used in newly-written code. Use gtkPaintPolygon instead.

Usage

gtkDrawPolygon(object, window, state.type, shadow.type, points, 
    fill)

Arguments

object

a GtkStyle

window

a GdkWindow

state.type

a state

shadow.type

type of shadow to draw

points

a list of GdkPoints

fill

TRUE if the polygon should be filled
