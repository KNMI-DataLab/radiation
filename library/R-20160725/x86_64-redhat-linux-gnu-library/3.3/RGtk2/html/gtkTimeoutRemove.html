gtkTimeoutRemove

Description

Removes the given timeout destroying all information about it. WARNING: gtk_timeout_remove has been deprecated since version 2.4 and should not be used in newly-written code. Use gSourceRemove instead.

Usage

gtkTimeoutRemove(timeout.handler.id)

Arguments

timeout.handler.id

The identifier returned when installing the timeout.
