gtkDrawExtension

Description

Draws an extension, i.e. a notebook tab. WARNING: gtk_draw_extension has been deprecated since version 2.0 and should not be used in newly-written code. Use gtkPaintExtension instead.

Usage

gtkDrawExtension(object, window, state.type, shadow.type, x, 
    y, width, height, gap.side)

Arguments

object

a GtkStyle

window

a GdkWindow

state.type

a state

shadow.type

type of shadow to draw

x

x origin of the extension

y

y origin of the extension

width

width of the extension

height

width of the extension

gap.side

the side on to which the extension is attached
