gtkNotebookQueryTabLabelPacking

Description

Query the packing attributes for the tab label of the page containing child. WARNING: gtk_notebook_query_tab_label_packing has been deprecated since version 2.20 and should not be used in newly-written code. Modify the "tab-expand" and "tab-fill" child properties instead.

Usage

gtkNotebookQueryTabLabelPacking(object, child)

Arguments

object

a GtkNotebook

child

the page

Value

A list containing the following elements:

expand

location to store the expand value (or NULL)

fill

location to store the fill value (or NULL)

pack.type

location to store the pack_type (or NULL)
