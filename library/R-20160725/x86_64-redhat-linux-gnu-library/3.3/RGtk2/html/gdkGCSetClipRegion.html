gdkGCSetClipRegion

Description

Sets the clip mask for a graphics context from a region structure. The clip mask is interpreted relative to the clip origin. (See gdkGCSetClipOrigin).

Usage

gdkGCSetClipRegion(object, region)

Arguments

object

a GdkGC.

region

the GdkRegion.
