gtkMenuGetAccelGroup

Description

Gets the GtkAccelGroup which holds global accelerators for the menu. See gtkMenuSetAccelGroup.

Usage

gtkMenuGetAccelGroup(object)

Arguments

object

a GtkMenu.

Value

[GtkAccelGroup] the GtkAccelGroup associated with the menu.
