gtkEntryGetInnerBorder

Description

This function returns the entry's "inner-border" property. See gtkEntrySetInnerBorder for more information.

Usage

gtkEntryGetInnerBorder(object)

Arguments

object

a GtkEntry

Details

Since 2.10

Value
