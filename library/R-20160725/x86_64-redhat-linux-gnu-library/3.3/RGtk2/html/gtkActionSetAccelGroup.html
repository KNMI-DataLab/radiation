gtkActionSetAccelGroup

Description

Sets the GtkAccelGroup in which the accelerator for this action will be installed.

Usage

gtkActionSetAccelGroup(object, accel.group)

Arguments

object

the action object

accel.group

a GtkAccelGroup or NULL. [ allow-none ]
