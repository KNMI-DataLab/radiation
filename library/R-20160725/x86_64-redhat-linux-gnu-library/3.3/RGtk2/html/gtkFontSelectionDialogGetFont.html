gtkFontSelectionDialogGetFont

Description

Gets the currently-selected font. WARNING: gtk_font_selection_dialog_get_font has been deprecated since version 2.0 and should not be used in newly-written code. Use gtkFontSelectionDialogGetFontName instead.

Usage

gtkFontSelectionDialogGetFont(object)

Arguments

object

a GtkFontSelectionDialog

Value

[GdkFont] the GdkFont from the GtkFontSelection for the currently selected font in the dialog, or NULL if no font is selected
