gtkAccelGroupsActivate

Description

Finds the first accelerator in any GtkAccelGroup attached to object that matches accel.key and accel.mods, and activates that accelerator.

Usage

gtkAccelGroupsActivate(object, accel.key, accel.mods)

Arguments

object

the GObject, usually a GtkWindow, on which to activate the accelerator.

accel.key

accelerator keyval from a key event

accel.mods

keyboard state mask from a key event

Value

[logical] TRUE if an accelerator was activated and handled this keypress
