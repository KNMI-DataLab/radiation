gtkContainerClassFindChildProperty

Description

Finds a child property of a container class by name.

Usage

gtkContainerClassFindChildProperty(cclass, property.name)

Arguments

cclass

a GtkContainerClass

property.name

the name of the child property to find

Value

[GParamSpec] the GParamSpec of the child property or NULL if class has no child property with that name. [ allow-none ]
