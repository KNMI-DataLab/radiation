cairoUserFontFaceGetUnicodeToGlyphFunc

Usage

cairoUserFontFaceGetUnicodeToGlyphFunc(font.face)

Arguments

font.face

[CairoFontFace] A user font face

Details

Since 1.8

Value

[cairo_user_scaled_font_unicode_to_glyph_func_t] The unicode_to_glyph callback of font.face or NULL if none set or an error occurred.
