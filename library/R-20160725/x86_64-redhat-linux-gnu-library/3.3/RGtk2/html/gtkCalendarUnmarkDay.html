gtkCalendarUnmarkDay

Description

Removes the visual marker from a particular day.

Usage

gtkCalendarUnmarkDay(object, day)

Arguments

object

a GtkCalendar.

day

the day number to unmark between 1 and 31.

Value

[logical] TRUE, always
