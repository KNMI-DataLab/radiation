pangoFontGetCoverage

Description

Computes the coverage map for a given font and language tag.

Usage

pangoFontGetCoverage(object, language)

Arguments

object

[PangoFont] a PangoFont

language

[PangoLanguage] the language tag

Value

[PangoCoverage] a newly-allocated PangoCoverage object.
