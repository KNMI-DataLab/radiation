cairoPsGetLevels

Description

Used to retrieve the list of supported levels. See cairoPsSurfaceRestrictToLevel.

Usage

cairoPsGetLevels()

Details

Since 1.6

Value

A list containing the following elements:

levels

[CairoPsLevel] supported level list

nlevels

[integer] list length
