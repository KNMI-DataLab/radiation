gtkCListSetCellStyle

Description

Sets the style for the specified cell. WARNING: gtk_clist_set_cell_style is deprecated and should not be used in newly-written code.

Usage

gtkCListSetCellStyle(object, row, column, style)

Arguments

object

The GtkCList to affect.

row

The row of the cell.

column

The column of the cell.

style

A pointer to a GtkStyle structure.
