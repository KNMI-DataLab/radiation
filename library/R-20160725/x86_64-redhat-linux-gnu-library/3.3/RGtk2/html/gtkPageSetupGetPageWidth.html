gtkPageSetupGetPageWidth

Description

Returns the page width in units of unit.

Usage

gtkPageSetupGetPageWidth(object, unit)

Arguments

object

a GtkPageSetup

unit

the unit for the return value

Details

Note that this function takes orientation and margins into consideration. See gtkPageSetupGetPaperWidth. Since 2.10

Value

[numeric] the page width.
