gtkLabelNew

Description

Creates a new label with the given text inside it. You can pass NULL to get an empty label widget.

Usage

gtkLabelNew(str = NULL, show = TRUE)

Arguments

str

The text of the label

Value

[GtkWidget] the new GtkLabel
