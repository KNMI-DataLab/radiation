gtkCListColumnTitlesActive

Description

Causes all column title buttons to become active. This is the same as calling gtkCListColumnTitleActive for each column. WARNING: gtk_clist_column_titles_active is deprecated and should not be used in newly-written code.

Usage

gtkCListColumnTitlesActive(object)

Arguments

object

The GtkCList to affect.
