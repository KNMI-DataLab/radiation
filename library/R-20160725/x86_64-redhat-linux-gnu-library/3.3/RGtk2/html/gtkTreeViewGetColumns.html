gtkTreeViewGetColumns

Description

Returns a list of all the GtkTreeViewColumn s currently in tree.view.

Usage

gtkTreeViewGetColumns(object)

Arguments

object

A GtkTreeView

Value

[list] A list of GtkTreeViewColumn s. [ element-type GtkTreeViewColumn][ transfer container ]
