atkEditableTextCutText

Description

Copy text from start.pos up to, but not including end.pos to the clipboard and then delete from the widget.

Usage

atkEditableTextCutText(object, start.pos, end.pos)

Arguments

object

[AtkEditableText] an AtkEditableText

start.pos

[integer] start position

end.pos

[integer] end position
