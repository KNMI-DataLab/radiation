gtkTextIterSetOffset

Description

Sets iter to point to char.offset. char.offset counts from the start of the entire text buffer, starting with 0.

Usage

gtkTextIterSetOffset(object, char.offset)

Arguments

object

a GtkTextIter

char.offset

a character number
