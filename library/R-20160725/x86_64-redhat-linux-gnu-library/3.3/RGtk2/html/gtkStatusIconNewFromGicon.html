gtkStatusIconNewFromGicon

Description

Creates a status icon displaying a GIcon. If the icon is a themed icon, it will be updated when the theme changes.

Usage

gtkStatusIconNewFromGicon(icon)

Arguments

icon

a GIcon

Details

Since 2.14

Value

[GtkStatusIcon] a new GtkStatusIcon
