gVolumeMonitorGetMountForUuid

Description

Finds a GMount object by its UUID (see gMountGetUuid)

Usage

gVolumeMonitorGetMountForUuid(object, uuid)

Arguments

object

a GVolumeMonitor.

uuid

the UUID to look for

Value

[GMount] a GMount or NULL if no such mount is available.
