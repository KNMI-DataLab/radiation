gdkDisplayGetEvent

Description

Gets the next GdkEvent to be processed for display, fetching events from the windowing system if necessary.

Usage

gdkDisplayGetEvent(object)

Arguments

object

a GdkDisplay

Details

Since 2.2

Value

[GdkEvent] the next GdkEvent to be processed, or NULL if no events are pending.
