gtkToolPaletteGetDragItem

Description

Get the dragged item from the selection. This could be a GtkToolItem or a GtkToolItemGroup.

Usage

gtkToolPaletteGetDragItem(object, selection)

Arguments

object

a GtkToolPalette

selection

a GtkSelectionData

Details

Since 2.20

Value

[GtkWidget] the dragged item in selection
