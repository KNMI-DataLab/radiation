gdkFontId

Description

Returns the X Font ID for the given font. WARNING: gdk_font_id is deprecated and should not be used in newly-written code.

Usage

gdkFontId(object)

Arguments

object

a GdkFont.

Value

[integer] the numeric X Font ID
