gtkTextViewSetPixelsInsideWrap

Description

Sets the default number of pixels of blank space to leave between display/wrapped lines within a paragraph. May be overridden by tags in text.view's buffer.

Usage

gtkTextViewSetPixelsInsideWrap(object, pixels.inside.wrap)

Arguments

object

a GtkTextView

pixels.inside.wrap

default number of pixels between wrapped lines
