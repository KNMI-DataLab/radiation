gtkRecentManagerSetScreen

Description

Sets the screen for a recent manager; the screen is used to track the user's currently configured recently used documents storage. WARNING: gtk_recent_manager_set_screen has been deprecated since version 2.12 and should not be used in newly-written code. This function has been deprecated and should not be used in newly written code. Calling this function has no effect.

Usage

gtkRecentManagerSetScreen(object, screen)

Arguments

object

a GtkRecentManager

screen

a GdkScreen
