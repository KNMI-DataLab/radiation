gAppInfoCanRemoveSupportsType

Description

Checks if a supported content type can be removed from an application.

Usage

gAppInfoCanRemoveSupportsType(object)

Arguments

object

a GAppInfo.

Value

[logical] TRUE if it is possible to remove supported content types from a given appinfo, FALSE if not.
