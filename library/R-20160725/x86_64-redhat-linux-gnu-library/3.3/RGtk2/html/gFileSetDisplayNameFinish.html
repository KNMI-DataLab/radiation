gFileSetDisplayNameFinish

Description

Finishes setting a display name started with gFileSetDisplayNameAsync.

Usage

gFileSetDisplayNameFinish(object, res, .errwarn = TRUE)

Arguments

object

input GFile.

res

a GAsyncResult.

.errwarn

Whether to issue a warning on error or fail silently

Value

A list containing the following elements:

retval

[GFile] a GFile or NULL on error.

error

a GError, or NULL
