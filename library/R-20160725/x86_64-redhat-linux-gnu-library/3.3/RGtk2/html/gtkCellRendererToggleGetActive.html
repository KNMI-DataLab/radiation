gtkCellRendererToggleGetActive

Description

Returns whether the cell renderer is active. See gtkCellRendererToggleSetActive.

Usage

gtkCellRendererToggleGetActive(object)

Arguments

object

a GtkCellRendererToggle

Value

[logical] TRUE if the cell renderer is active.
