gtkRadioToolButtonNewFromStock

Description

Creates a new GtkRadioToolButton, adding it to group. The new GtkRadioToolButton will contain an icon and label from the stock item indicated by stock.id.

Usage

gtkRadioToolButtonNewFromStock(group = NULL, stock.id, show = TRUE)

Arguments

group

an existing radio button group, or NULL if you are creating a new group. [ allow-none ]

stock.id

the name of a stock item

Details

Since 2.4

Value

[GtkToolItem] The new GtkRadioToolItem
