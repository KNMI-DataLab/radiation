gtkPreviewSetDither

Description

Set the dithering mode for the display. WARNING: gtk_preview_set_dither is deprecated and should not be used in newly-written code.

Usage

gtkPreviewSetDither(object, dither)

Arguments

object

a GtkPreview.

dither

the dithering mode.
