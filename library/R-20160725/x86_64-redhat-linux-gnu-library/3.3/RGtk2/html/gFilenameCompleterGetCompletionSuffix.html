gFilenameCompleterGetCompletionSuffix

Description

Obtains a completion for initial.text from completer.

Usage

gFilenameCompleterGetCompletionSuffix(object, initial.text)

Arguments

object

the filename completer.

initial.text

text to be completed.

Value

[char] a completed string, or NULL if no completion exists.
