gtkTextAttributesCopyValues

Description

Copies the values from src to dest so that dest has the same values as src. Frees existing values in dest.

Usage

gtkTextAttributesCopyValues(object, dest)

Arguments

object

a GtkTextAttributes

dest

another GtkTextAttributes
