gtkVButtonBoxGetLayoutDefault

Description

Retrieves the current layout used to arrange buttons in button box widgets. WARNING: gtk_vbutton_box_get_layout_default is deprecated and should not be used in newly-written code.

Usage

gtkVButtonBoxGetLayoutDefault()

Value

[GtkButtonBoxStyle] the current GtkButtonBoxStyle.
