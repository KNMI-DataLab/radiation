gtkDrawHline

Description

Draws a horizontal line from (x1, y) to (x2, y) in window using the given style and state. WARNING: gtk_draw_hline has been deprecated since version 2.0 and should not be used in newly-written code. Use gtkPaintHline instead.

Usage

gtkDrawHline(object, window, state.type, x1, x2, y)

Arguments

object

a GtkStyle

window

a GdkWindow

state.type

a state

x1

the starting x coordinate

x2

the ending x coordinate

y

the y coordinate
