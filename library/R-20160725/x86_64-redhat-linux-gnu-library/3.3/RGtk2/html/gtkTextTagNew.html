gtkTextTagNew

Description

Creates a GtkTextTag. Configure the tag using object arguments, i.e. using gObjectSet.

Usage

gtkTextTagNew(name = NULL)

Arguments

name

tag name, or NULL. [ allow-none ]

Value

[GtkTextTag] a new GtkTextTag
