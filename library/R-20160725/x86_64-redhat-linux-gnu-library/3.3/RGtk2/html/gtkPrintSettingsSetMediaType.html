gtkPrintSettingsSetMediaType

Description

Sets the value of GTK_PRINT_SETTINGS_MEDIA_TYPE.

Usage

gtkPrintSettingsSetMediaType(object, media.type)

Arguments

object

a GtkPrintSettings

media.type

the media type

Details

The set of media types is defined in PWG 5101.1-2002 PWG.
