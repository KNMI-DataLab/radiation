gtkFixedPut

Description

Adds a widget to a GtkFixed container at the given position.

Usage

gtkFixedPut(object, widget, x, y)

Arguments

object

a GtkFixed.

widget

the widget to add.

x

the horizontal position to place the widget at.

y

the vertical position to place the widget at.
