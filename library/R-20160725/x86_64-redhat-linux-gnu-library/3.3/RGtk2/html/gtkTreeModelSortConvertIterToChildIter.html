gtkTreeModelSortConvertIterToChildIter

Description

Sets child.iter to point to the row pointed to by sorted.iter.

Usage

gtkTreeModelSortConvertIterToChildIter(object, sorted.iter)

Arguments

object

A GtkTreeModelSort

sorted.iter

A valid GtkTreeIter pointing to a row on tree.model.sort.

Value

A list containing the following elements:

child.iter

An uninitialized GtkTreeIter
