Usage

  ## S4 method for signature 'NMFStrategyOctave'
algorithm(object,
    load = FALSE)

Arguments

load

logical that indicates if the algorithm should be loaded as an R function.

object

an object computed using some algorithm, or that describes an algorithm itself.
