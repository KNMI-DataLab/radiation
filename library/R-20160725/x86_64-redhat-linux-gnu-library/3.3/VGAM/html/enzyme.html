Enzyme Data

Description

Enzyme velocity and substrate concentration.

Usage

data(enzyme)

Format

A data frame with 12 observations on the following 2 variables.

conc

a numeric explanatory vector; substrate concentration

velocity

a numeric response vector; enzyme velocity
