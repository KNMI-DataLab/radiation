gtkPreviewGetInfo

Description

Return a GtkPreviewInfo structure containing global information about preview widgets. WARNING: gtk_preview_get_info is deprecated and should not be used in newly-written code.

Usage

gtkPreviewGetInfo()

Value

[GtkPreviewInfo] a GtkPreviewInfo structure. The return value belongs to GTK+ and must not be modified or freed.
