gtkStatusIconNewFromFile

Description

Creates a status icon displaying the file filename.

Usage

gtkStatusIconNewFromFile(filename)

Arguments

filename

a filename

Details

The image will be scaled down to fit in the available space in the notification area, if necessary. Since 2.10

Value

[GtkStatusIcon] a new GtkStatusIcon
