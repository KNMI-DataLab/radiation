gLoadableIconLoadAsync

Description

Loads an icon asynchronously. To finish this function, see gLoadableIconLoadFinish. For the synchronous, blocking version of this function, see gLoadableIconLoad.

Usage

gLoadableIconLoadAsync(object, size, cancellable = NULL, callback, 
    user.data = NULL)

Arguments

object

a GLoadableIcon.

size

an integer.

cancellable

optional GCancellable object, NULL to ignore.

callback

a GAsyncReadyCallback to call when the request is satisfied

user.data

the data to pass to callback function
