gtkPrintOperationSetCurrentPage

Description

Sets the current page.

Usage

gtkPrintOperationSetCurrentPage(object, current.page)

Arguments

object

a GtkPrintOperation

current.page

the current page, 0-based

Details

If this is called before gtkPrintOperationRun, the user will be able to select to print only the current page.

Note that this only makes sense for pre-paginated documents. Since 2.10
