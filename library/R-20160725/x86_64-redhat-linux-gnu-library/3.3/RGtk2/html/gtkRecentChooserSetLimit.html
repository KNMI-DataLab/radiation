gtkRecentChooserSetLimit {RGtk2}R Documentation

gtkRecentChooserSetLimit

Description

Sets the number of items that should be returned by gtkRecentChooserGetItems and gtkRecentChooserGetUris.

Usage

gtkRecentChooserSetLimit(object, limit)

Arguments

object

a GtkRecentChooser

limit

a positive integer, or -1 for all items
