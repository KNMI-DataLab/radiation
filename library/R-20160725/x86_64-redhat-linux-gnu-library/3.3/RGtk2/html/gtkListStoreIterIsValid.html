gtkListStoreIterIsValid

Description

WARNING:

Usage

gtkListStoreIterIsValid(object, iter)

Arguments

object

A GtkListStore.

iter

A GtkTreeIter.

Details

Checks if the given iter is a valid iter for this GtkListStore. Since 2.2

Value

[logical] TRUE if the iter is valid, FALSE if the iter is invalid.
