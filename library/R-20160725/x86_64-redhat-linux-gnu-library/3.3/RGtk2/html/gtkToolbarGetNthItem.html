gtkToolbarGetNthItem

Description

Returns the n'th item on toolbar, or NULL if the toolbar does not contain an n'th item.

Usage

gtkToolbarGetNthItem(object, n)

Arguments

object

a GtkToolbar

n

A position on the toolbar

Details

Since 2.4

Value

[GtkToolItem] The n'th GtkToolItem on toolbar, or NULL if there isn't an n'th item.
