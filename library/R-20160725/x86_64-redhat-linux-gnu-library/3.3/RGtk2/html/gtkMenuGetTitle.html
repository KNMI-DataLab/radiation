gtkMenuGetTitle

Description

Returns the title of the menu. See gtkMenuSetTitle.

Usage

gtkMenuGetTitle(object)

Arguments

object

a GtkMenu

Value

[character] the title of the menu, or NULL if the menu has no title set on it.
