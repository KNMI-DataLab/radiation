gtkMenuItemGetAccelPath

Description

Retrieve the accelerator path that was previously set on menu.item.

Usage

gtkMenuItemGetAccelPath(object)

Arguments

object

a valid GtkMenuItem

Details

See gtkMenuItemSetAccelPath for details. Since 2.14

Value

[character] the accelerator path corresponding to this menu item's functionality, or NULL if not set
