gtkButtonEnter

Description

Emits a gtkButtonEnter signal to the given GtkButton. WARNING: gtk_button_enter has been deprecated since version 2.20 and should not be used in newly-written code. Use the "enter-notify-event" signal.

Usage

gtkButtonEnter(object)

Arguments

object

The GtkButton you want to send the signal to.
