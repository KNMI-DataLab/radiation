gtkNotebookSetTabPos

Description

Sets the edge at which the tabs for switching pages in the notebook are drawn.

Usage

gtkNotebookSetTabPos(object, pos)

Arguments

object

a GtkNotebook.

pos

the edge to draw the tabs at.
