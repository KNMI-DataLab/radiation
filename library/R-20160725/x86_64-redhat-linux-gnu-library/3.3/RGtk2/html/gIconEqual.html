gIconEqual

Description

Checks if two icons are equal.

Usage

gIconEqual(object, icon2)

Arguments

object

pointer to the first GIcon.

icon2

pointer to the second GIcon.

Value

[logical] TRUE if icon1 is equal to icon2. FALSE otherwise.
