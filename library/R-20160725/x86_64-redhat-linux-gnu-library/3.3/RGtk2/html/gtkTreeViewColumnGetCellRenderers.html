gtkTreeViewColumnGetCellRenderers

Description

Returns a newly-allocated list of all the cell renderers in the column, in no particular order. WARNING: gtk_tree_view_column_get_cell_renderers has been deprecated since version 2.18 and should not be used in newly-written code. use gtkCellLayoutGetCells instead.

Usage

gtkTreeViewColumnGetCellRenderers(object)

Arguments

object

A GtkTreeViewColumn

Value

[list] A list of GtkCellRenderers
