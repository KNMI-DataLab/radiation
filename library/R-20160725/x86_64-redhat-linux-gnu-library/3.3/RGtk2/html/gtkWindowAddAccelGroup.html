gtkWindowAddAccelGroup

Description

Associate accel.group with window, such that calling gtkAccelGroupsActivate on window will activate accelerators in accel.group.

Usage

gtkWindowAddAccelGroup(object, accel.group)

Arguments

object

window to attach accelerator group to

accel.group

a GtkAccelGroup
