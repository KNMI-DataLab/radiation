gtkMessageDialogFormatSecondaryText

Description

Sets the secondary text of the message dialog to be message.format (with printf()-style).

Usage

gtkMessageDialogFormatSecondaryText(object, ...)

Arguments

object

a GtkMessageDialog

...

undocumented

Details

Note that setting a secondary text makes the primary text become bold, unless you have provided explicit markup. Since 2.6
