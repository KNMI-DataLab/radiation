gtkMessageDialogSetMarkup

Description

Sets the text of the message dialog to be str, which is marked up with the Pango text markup language.

Usage

gtkMessageDialogSetMarkup(object, str)

Arguments

object

a GtkMessageDialog

str

markup string (see Pango markup format)
