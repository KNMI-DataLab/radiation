gtkCalendarSelectMonth

Description

Shifts the calendar to a different month.

Usage

gtkCalendarSelectMonth(object, month, year)

Arguments

object

a GtkCalendar

month

a month number between 0 and 11.

year

the year the month is in.

Value

[logical] TRUE, always
