gInetAddressNewAny

Description

Creates a GInetAddress for the "any" address (unassigned/"don't care") for family.

Usage

gInetAddressNewAny(family)

Arguments

family

the address family

Details

Since 2.22

Value

[GInetAddress] a new GInetAddress corresponding to the "any" address for family.
