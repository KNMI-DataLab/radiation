gtkRcStyleCopy

Description

Makes a copy of the specified GtkRcStyle. This function will correctly copy an RC style that is a member of a class derived from GtkRcStyle.

Usage

gtkRcStyleCopy(object)

Arguments

object

the style to copy

Value

[GtkRcStyle] the resulting GtkRcStyle
