gtkWidgetState

Description

Get the states of a widget.

Usage

gtkWidgetState(wid)
gtkWidgetSavedState(wid)

Arguments

wid

The widget to query for its (saved) state.

Value

A state (GtkStateType) of the widget, either saved or current.
