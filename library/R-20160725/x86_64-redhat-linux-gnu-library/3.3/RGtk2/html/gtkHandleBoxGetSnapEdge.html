gtkHandleBoxGetSnapEdge

Description

Gets the edge used for determining reattachment of the handle box. See gtkHandleBoxSetSnapEdge.

Usage

gtkHandleBoxGetSnapEdge(object)

Arguments

object

a GtkHandleBox

Value

[GtkPositionType] the edge used for determining reattachment, or (GtkPositionType)-1 if this is determined (as per default) from the handle position.
