gtkPrintOperationGetPrintSettings

Description

Returns the current print settings.

Usage

gtkPrintOperationGetPrintSettings(object)

Arguments

object

a GtkPrintOperation

Details

Note that the return value is NULL until either gtkPrintOperationSetPrintSettings or gtkPrintOperationRun have been called. Since 2.10

Value

[GtkPrintSettings] the current print settings of op.
