pangoAttrTypeGetName

Description

Fetches the attribute type name passed in when registering the type using pangoAttrTypeRegister.

Usage

pangoAttrTypeGetName(type)

Arguments

type

[PangoAttrType] an attribute type ID to fetch the name for

Details

The returned value is an interned string (see gInternString() for what that means) that should not be modified or freed. Since 1.22

Value

[char] the type ID name (which may be NULL), or NULL if type is a built-in Pango attribute type or invalid.
