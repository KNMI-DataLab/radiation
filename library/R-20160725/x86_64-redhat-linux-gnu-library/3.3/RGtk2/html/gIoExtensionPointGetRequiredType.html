gIoExtensionPointGetRequiredType

Description

Gets the required type for extension.point.

Usage

gIoExtensionPointGetRequiredType(object)

Arguments

object

a GIOExtensionPoint

Value

[GType] the GType that all implementations must have, or G_TYPE_INVALID if the extension point has no required type
