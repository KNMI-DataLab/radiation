Detailed Description

These functions are used to create and destroy cursors. There is a number of standard cursors, but it is also possible to construct new cursors from pixmaps and pixbufs. There may be limitations as to what kinds of cursors can be constructed on a given display, see gdkDisplaySupportsCursorAlpha, gdkDisplaySupportsCursorColor, gdkDisplayGetDefaultCursorSize and gdkDisplayGetMaximalCursorSize.

Cursors by themselves are not very interesting, they must be be bound to a window for users to see them. This is done with gdkWindowSetCursor or by setting the cursor member of the GdkWindowAttr struct passed to gdkWindowNew.
