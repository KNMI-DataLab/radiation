gtkAccelMapLookupEntry

Description

Looks up the accelerator entry for accel.path and fills in key.

Usage

gtkAccelMapLookupEntry(accel.path)

Arguments

accel.path

a valid accelerator path

Value

A list containing the following elements:

retval

[logical] TRUE if accel.path is known, FALSE otherwise

key

the accelerator key to be filled in (optional)
