gtkAssistantSetPageHeaderImage

Description

Sets a header image for page. This image is displayed in the header area of the assistant when page is the current page.

Usage

gtkAssistantSetPageHeaderImage(object, page, pixbuf = NULL)

Arguments

object

a GtkAssistant

page

a page of assistant

pixbuf

the new header image page. [ allow-none ]
