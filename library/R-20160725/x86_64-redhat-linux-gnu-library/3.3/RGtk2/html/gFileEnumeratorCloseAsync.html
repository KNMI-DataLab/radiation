gFileEnumeratorCloseAsync

Description

Asynchronously closes the file enumerator.

Usage

gFileEnumeratorCloseAsync(object, io.priority = 0, cancellable = NULL, 
    callback, user.data = NULL)

Arguments

object

a GFileEnumerator.

io.priority

the I/O priority of the request.

cancellable

optional GCancellable object, NULL to ignore.

callback

a GAsyncReadyCallback to call when the request is satisfied

user.data

the data to pass to callback function

Details

If cancellable is not NULL, then the operation can be cancelled by triggering the cancellable object from another thread. If the operation was cancelled, the error G_IO_ERROR_CANCELLED will be returned in gFileEnumeratorCloseFinish.
