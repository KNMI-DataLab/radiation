atkTableGetNRows

Description

Gets the number of rows in the table.

Usage

atkTableGetNRows(object)

Arguments

object

[AtkTable] a GObject instance that implements AtkTableIface

Value

[integer] a gint representing the number of rows, or 0 if value does not implement this interface.
