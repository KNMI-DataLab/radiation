gdkKeyvalFromName

Description

Converts a key name to a key value.

Usage

gdkKeyvalFromName(keyval.name)

Arguments

keyval.name

a key name.

Value

[numeric] the corresponding key value, or GDK_VoidSymbol if the key name is not a valid key.
