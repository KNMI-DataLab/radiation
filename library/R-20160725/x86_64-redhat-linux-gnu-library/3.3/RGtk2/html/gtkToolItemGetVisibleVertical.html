gtkToolItemGetVisibleVertical

Description

Returns whether tool.item is visible when the toolbar is docked vertically. See gtkToolItemSetVisibleVertical.

Usage

gtkToolItemGetVisibleVertical(object)

Arguments

object

a GtkToolItem

Details

Since 2.4

Value

[logical] Whether tool.item is visible when the toolbar is docked vertically
