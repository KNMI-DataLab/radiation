gResolverLookupByAddressAsync

Description

Begins asynchronously reverse-resolving address to determine its associated hostname, and eventually calls callback, which must call gResolverLookupByAddressFinish to get the final result.

Usage

gResolverLookupByAddressAsync(object, address, cancellable = NULL, 
    callback, user.data = NULL)

Arguments

object

a GResolver

address

the address to reverse-resolve

cancellable

a GCancellable, or NULL

callback

callback to call after resolution completes

user.data

data for callback
