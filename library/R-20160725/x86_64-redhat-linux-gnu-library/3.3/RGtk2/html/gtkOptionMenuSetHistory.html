gtkOptionMenuSetHistory

Description

Selects the menu item specified by index. making it the newly selected value for the option menu. WARNING: gtk_option_menu_set_history has been deprecated since version 2.4 and should not be used in newly-written code. Use GtkComboBox instead.

Usage

gtkOptionMenuSetHistory(object, index)

Arguments

object

a GtkOptionMenu.

index

the index of the menu item to select. Index values are from 0 to n-1.
