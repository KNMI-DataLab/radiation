atkTableGetRowHeader

Description

Gets the row header of a specified row in an accessible table.

Usage

atkTableGetRowHeader(object, row)

Arguments

object

[AtkTable] a GObject instance that implements AtkTableIface

row

[integer] a integer representing a row in the table

Value

[AtkObject] a AtkObject* representing the specified row header, or NULL if value does not implement this interface.
