pangoContextGetFontDescription

Description

Retrieve the default font description for the context.

Usage

pangoContextGetFontDescription(object)

Arguments

object

[PangoContext] a PangoContext

Value

[PangoFontDescription] a pointer to the context's default font description. This value must not be modified or freed.
