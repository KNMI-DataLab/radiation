atkImplementorRefAccessible

Description

Gets a reference to an object's AtkObject implementation, if the object implements AtkObjectIface

Usage

atkImplementorRefAccessible(object)

Arguments

object

[AtkImplementor] The GObject instance which should implement AtkImplementorIface if a non-null return value is required.

Value

[AtkObject] a reference to an object's AtkObject implementation
