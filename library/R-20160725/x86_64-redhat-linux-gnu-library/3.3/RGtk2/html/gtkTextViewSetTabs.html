gtkTextViewSetTabs

Description

Sets the default tab stops for paragraphs in text.view. Tags in the buffer may override the default.

Usage

gtkTextViewSetTabs(object, tabs)

Arguments

object

a GtkTextView

tabs

tabs as a PangoTabArray
