gtkRcPropertyParseRequisition

Description

A GtkRcPropertyParser for use with gtkSettingsInstallPropertyParser or gtkWidgetClassInstallStylePropertyParser which parses a requisition in the form "{ width, height }" for integers width and height.

Usage

gtkRcPropertyParseRequisition(pspec, gstring)

Arguments

pspec

a GParamSpec

gstring

the character to be parsed

Value

A list containing the following elements:

retval

[logical] TRUE if gstring could be parsed and property.value has been set to the resulting GtkRequisition.

property.value

a R object which must hold boxed values.
