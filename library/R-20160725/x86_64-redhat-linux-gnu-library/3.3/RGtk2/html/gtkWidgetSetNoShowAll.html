gtkWidgetSetNoShowAll

Description

Sets the "no-show-all" property, which determines whether calls to gtkWidgetShowAll and gtkWidgetHideAll will affect this widget.

Usage

gtkWidgetSetNoShowAll(object, no.show.all)

Arguments

object

a GtkWidget

no.show.all

the new value for the "no-show-all" property

Details

This is mostly for use in constructing widget hierarchies with externally controlled visibility, see GtkUIManager. Since 2.4
