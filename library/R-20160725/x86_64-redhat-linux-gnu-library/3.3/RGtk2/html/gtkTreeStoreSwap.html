gtkTreeStoreSwap

Description

Swaps a and b in the same level of tree.store. Note that this function only works with unsorted stores.

Usage

gtkTreeStoreSwap(object, a, b)

Arguments

object

A GtkTreeStore.

a

A GtkTreeIter.

b

Another GtkTreeIter.
