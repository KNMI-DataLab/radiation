gtkListSelectItem

Description

Selects the child number item of the list. Nothing happens if item is out of bounds. The signal GtkList::select-child will be emitted. WARNING: gtk_list_select_item is deprecated and should not be used in newly-written code.

Usage

gtkListSelectItem(object, item)

Arguments

object

the list widget.

item

the index of the child to select.
