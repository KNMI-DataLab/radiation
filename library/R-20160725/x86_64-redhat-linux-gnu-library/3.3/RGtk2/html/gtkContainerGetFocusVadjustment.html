gtkContainerGetFocusVadjustment

Description

Retrieves the vertical focus adjustment for the container. See gtkContainerSetFocusVadjustment.

Usage

gtkContainerGetFocusVadjustment(object)

Arguments

object

a GtkContainer

Value

[GtkAdjustment] the vertical focus adjustment, or NULL if none has been set. [ transfer none ]
