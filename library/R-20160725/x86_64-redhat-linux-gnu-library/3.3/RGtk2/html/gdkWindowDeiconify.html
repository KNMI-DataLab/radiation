gdkWindowDeiconify

Description

Attempt to deiconify (unminimize) window. On X11 the window manager may choose to ignore the request to deiconify. When using GTK+, use gtkWindowDeiconify instead of the GdkWindow variant. Or better yet, you probably want to use gtkWindowPresent, which raises the window, focuses it, unminimizes it, and puts it on the current desktop.

Usage

gdkWindowDeiconify(object)

Arguments

object

a toplevel GdkWindow
