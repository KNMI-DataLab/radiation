gtkColorSelectionSetHasPalette

Description

Shows and hides the palette based upon the value of has.palette.

Usage

gtkColorSelectionSetHasPalette(object, has.palette)

Arguments

object

a GtkColorSelection.

has.palette

TRUE if palette is to be visible, FALSE otherwise.
