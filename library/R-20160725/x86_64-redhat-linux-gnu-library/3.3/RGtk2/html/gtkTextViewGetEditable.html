gtkTextViewGetEditable

Description

Returns the default editability of the GtkTextView. Tags in the buffer may override this setting for some ranges of text.

Usage

gtkTextViewGetEditable(object)

Arguments

object

a GtkTextView

Value

[logical] whether text is editable by default
