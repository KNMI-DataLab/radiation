gBufferedInputStreamNewSized

Description

Creates a new GBufferedInputStream from the given base.stream, with a buffer set to size.

Usage

gBufferedInputStreamNewSized(base.stream, size)

Arguments

base.stream

a GInputStream.

size

a numeric.

Value

[GInputStream] a GInputStream.
