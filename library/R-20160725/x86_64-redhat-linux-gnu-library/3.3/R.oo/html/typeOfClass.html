Description

Gets the type of a class (S3 or S4).

Usage

## Default S3 method:
typeOfClass(object, ...)

Arguments

object

The object to be checks.

...

Not used.

Value

Returns a character string "S3", "S3-Object" or "S4", or NA if neither.
