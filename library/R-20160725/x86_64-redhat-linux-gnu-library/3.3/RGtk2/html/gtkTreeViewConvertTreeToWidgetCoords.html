gtkTreeViewConvertTreeToWidgetCoords

Description

Converts tree coordinates (coordinates in full scrollable area of the tree) to widget coordinates.

Usage

gtkTreeViewConvertTreeToWidgetCoords(object, tx, ty)

Arguments

object

a GtkTreeView

tx

X coordinate relative to the tree

ty

Y coordinate relative to the tree

Details

Since 2.12

Value

A list containing the following elements:

wx

return location for widget X coordinate

wy

return location for widget Y coordinate
