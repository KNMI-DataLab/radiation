window_chron {StreamMetabolism}R Documentation

Time Windows of Diurnal Curves

Description

Takes a time window of a larger series

Usage

window_chron(x, day1, hour1, day2, hour2, ...)

Arguments

x

data to be subsetted

day1

start day

hour1

start time

day2

end date

hour2

end time

...

other arguments

Value

subset by time

Author(s)

Stephen A Sefick Jr.

References

chron, window, window.zoo

See Also

window

Examples

#with real data
data(DOTemp)
d <- window_chron(DOTemp, "8/18/70", "06:00:00", "8/18/70", "20:15:00")
plot(d)
