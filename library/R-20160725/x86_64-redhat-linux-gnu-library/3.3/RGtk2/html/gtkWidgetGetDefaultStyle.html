gtkWidgetGetDefaultStyle

Description

Returns the default style used by all widgets initially.

Usage

gtkWidgetGetDefaultStyle()

Value

[GtkStyle] the default style. This GtkStyle object is owned by GTK+ and should not be modified or freed. [ transfer none ]
