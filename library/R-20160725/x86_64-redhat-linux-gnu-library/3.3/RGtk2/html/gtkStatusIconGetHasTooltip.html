gtkStatusIconGetHasTooltip

Description

Returns the current value of the has-tooltip property. See "has-tooltip" for more information.

Usage

gtkStatusIconGetHasTooltip(object)

Arguments

object

a GtkStatusIcon

Details

Since 2.16

Value

[logical] current value of has-tooltip on status.icon.
