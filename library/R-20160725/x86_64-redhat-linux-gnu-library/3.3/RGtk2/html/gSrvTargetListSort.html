gSrvTargetListSort

Description

Sorts targets in place according to the algorithm in RFC 2782.

Usage

gSrvTargetListSort(targets)

Arguments

targets

a list of GSrvTarget

Details

Since 2.22

Value

[list] the head of the sorted list.
