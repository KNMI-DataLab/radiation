gtkPaintFlatBox

Description

Draws a flat box on window with the given parameters.

Usage

gtkPaintFlatBox(object, window, state.type, shadow.type, area = NULL, 
    widget = NULL, detail = NULL, x, y, width, height)

Arguments

object

a GtkStyle

window

a GdkWindow

state.type

a state

shadow.type

the type of shadow to draw

area

clip rectangle, or NULL if the output should not be clipped. [ allow-none ]

widget

the widget. [ allow-none ]

detail

a style detail. [ allow-none ]

x

x origin of the box

y

y origin of the box

width

the width of the box

height

the height of the box
