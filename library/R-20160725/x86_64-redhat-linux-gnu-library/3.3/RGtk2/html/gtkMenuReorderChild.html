gtkMenuReorderChild

Description

Moves a GtkMenuItem to a new position within the GtkMenu.

Usage

gtkMenuReorderChild(object, child, position)

Arguments

object

a GtkMenu.

child

the GtkMenuItem to move.

position

the new position to place child. Positions are numbered from 0 to n-1.
