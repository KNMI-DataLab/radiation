gtkCTreeNew

Description

Create a new GtkCTree widget. WARNING: gtk_ctree_new is deprecated and should not be used in newly-written code.

Usage

gtkCTreeNew(columns = 1, tree.column = 0, show = TRUE)

Arguments

columns

Number of columns.

tree.column

Which columns has the tree graphic.

Value

[GtkWidget] The new GtkCTree widget.
