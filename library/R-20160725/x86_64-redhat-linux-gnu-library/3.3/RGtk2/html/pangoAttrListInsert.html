pangoAttrListInsert

Description

Insert the given attribute into the PangoAttrList. It will be inserted after all other attributes with a matching start.index.

Usage

pangoAttrListInsert(object, attr)

Arguments

object

[PangoAttrList] a PangoAttrList

attr

[PangoAttribute] the attribute to insert. Ownership of this value is assumed by the list.
