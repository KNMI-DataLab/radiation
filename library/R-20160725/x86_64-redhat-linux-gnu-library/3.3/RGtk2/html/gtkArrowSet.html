gtkArrowSet

Description

Sets the direction and style of the GtkArrow, arrow.

Usage

gtkArrowSet(object, arrow.type, shadow.type)

Arguments

object

a widget of type GtkArrow.

arrow.type

a valid GtkArrowType.

shadow.type

a valid GtkShadowType.
