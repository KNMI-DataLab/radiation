gtkNotebookSetTabReorderable

Description

Sets whether the notebook tab can be reordered via drag and drop or not.

Usage

gtkNotebookSetTabReorderable(object, child, reorderable)

Arguments

object

a GtkNotebook

child

a child GtkWidget

reorderable

whether the tab is reorderable or not.
