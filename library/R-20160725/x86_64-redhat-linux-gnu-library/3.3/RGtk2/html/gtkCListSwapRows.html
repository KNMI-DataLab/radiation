gtkCListSwapRows

Description

Swaps the two specified rows with each other. WARNING: gtk_clist_swap_rows is deprecated and should not be used in newly-written code.

Usage

gtkCListSwapRows(object, row1, row2)

Arguments

object

The GtkCList to affect.

row1

Number of the first row.

row2

Number of the second row.
