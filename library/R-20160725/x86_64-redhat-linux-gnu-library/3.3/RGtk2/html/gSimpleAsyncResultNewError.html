gSimpleAsyncResultNewError

Description

Creates a new GSimpleAsyncResult with a set error.

Usage

gSimpleAsyncResultNewError(source.object, callback, user.data, 
    domain, code, format, ...)

Arguments

source.object

a GObject, or NULL.

callback

a GAsyncReadyCallback.

user.data

user data passed to callback.

domain

a GQuark.

code

an error code.

format

a string with format characters.

...

a list of values to insert into format.

Value

[GSimpleAsyncResult] a GSimpleAsyncResult.
