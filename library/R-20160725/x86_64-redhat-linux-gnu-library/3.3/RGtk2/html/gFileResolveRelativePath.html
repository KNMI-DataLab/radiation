gFileResolveRelativePath

Description

Resolves a relative path for file to an absolute path.

Usage

gFileResolveRelativePath(object, relative.path)

Arguments

object

input GFile.

relative.path

a given relative path string.

Details

This call does no blocking i/o.

Value

[GFile] GFile to the resolved path. NULL if relative.path is NULL or if file is invalid.
