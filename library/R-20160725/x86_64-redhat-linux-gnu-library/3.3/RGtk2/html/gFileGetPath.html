gFileGetPath

Description

Gets the local pathname for GFile, if one exists.

Usage

gFileGetPath(object)

Arguments

object

input GFile.

Details

This call does no blocking i/o.

Value

[char] string containing the GFile's path, or NULL if no such path exists.
