quantil.emp {regpro}R Documentation

Empirical p:th quantile

Description

Computes the value of the empirical p:th quantile.

Usage

quantil.emp(y, p)

Arguments

y

n vector; the observed data

p

0<p<1: the p:th quantile will be estimated

Value

a real number

Author(s)

Jussi Klemela

See Also

emp.quantile,

Examples

set.seed(2)
n<-100
y<-matrix(runif(n),n,1)
quantil.emp(y,p=0.05)
