gtkEntryBufferNew

Description

Create a new GtkEntryBuffer object.

Usage

gtkEntryBufferNew(initial.chars = NULL, n.initial.chars = -1)

Arguments

initial.chars

initial buffer text, or NULL. [ allow-none ]

n.initial.chars

number of characters in initial.chars, or -1

Details

Optionally, specify initial text to set in the buffer. Since 2.18

Value

[GtkEntryBuffer] A new GtkEntryBuffer object.
