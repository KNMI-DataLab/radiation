gFileInfoSetAttributeInt64

Description

Sets the attribute to contain the given attr.value, if possible.

Usage

gFileInfoSetAttributeInt64(object, attribute, attr.value)

Arguments

object

a GFileInfo.

attribute

attribute name to set.

attr.value

int64 value to set attribute to.
