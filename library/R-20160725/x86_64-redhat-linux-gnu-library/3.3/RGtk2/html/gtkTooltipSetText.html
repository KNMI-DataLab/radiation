gtkTooltipSetText

Description

Sets the text of the tooltip to be text. If text is NULL, the label will be hidden. See also gtkTooltipSetMarkup.

Usage

gtkTooltipSetText(object, text)

Arguments

object

a GtkTooltip

text

a text string or NULL. [ allow-none ]
