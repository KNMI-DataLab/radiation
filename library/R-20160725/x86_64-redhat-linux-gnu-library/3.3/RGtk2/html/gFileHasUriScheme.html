gFileHasUriScheme

Description

Checks to see if a GFile has a given URI scheme.

Usage

gFileHasUriScheme(object, uri.scheme)

Arguments

object

input GFile.

uri.scheme

a string containing a URI scheme.

Details

This call does no blocking i/o.

Value

[logical] TRUE if GFile's backend supports the given URI scheme, FALSE if URI scheme is NULL, not supported, or GFile is invalid.
