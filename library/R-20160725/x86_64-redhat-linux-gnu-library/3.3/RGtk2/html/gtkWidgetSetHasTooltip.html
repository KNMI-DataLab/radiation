gtkWidgetSetHasTooltip

Description

Sets the has-tooltip property on widget to has.tooltip. See GtkWidget:has-tooltip for more information.

Usage

gtkWidgetSetHasTooltip(object, has.tooltip)

Arguments

object

a GtkWidget

has.tooltip

whether or not widget has a tooltip.
