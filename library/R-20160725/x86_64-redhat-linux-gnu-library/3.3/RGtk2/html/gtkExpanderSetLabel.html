gtkExpanderSetLabel

Description

Sets the text of the label of the expander to label.

Usage

gtkExpanderSetLabel(object, label = NULL)

Arguments

object

a GtkExpander

label

a string. [ allow-none ]

Details

This will also clear any previously set labels. Since 2.4
