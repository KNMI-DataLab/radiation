gtkWindowGetDefaultIconList

Description

Gets the value set by gtkWindowSetDefaultIconList. but the pixbufs in the list have not had their reference count incremented.

Usage

gtkWindowGetDefaultIconList()

Value

[list] copy of default icon list. [ element-type GdkPixbuf][ transfer container ]
