gFilterOutputStreamSetCloseBaseStream

Description

Sets whether the base stream will be closed when stream is closed.

Usage

gFilterOutputStreamSetCloseBaseStream(object, close.base)

Arguments

object

a GFilterOutputStream.

close.base

TRUE to close the base stream.
